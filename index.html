notif_rqt_system_Obj: object

proCode: string (up to 6 chars)

Return Process Code

Possible Value Definition
000000 Transaction Successful
proMsg: string (up to 128 chars)

Corresponding Text Message of Process Code

Example
{
  "proCode": "000000",
  "proMsg": "Transaction Successful"
}

notif_rqt_txn_Obj: object

txnRef: string (up to 25 chars)

Returning Transaction Reference defined by Merchant

bankTxnId: string (up to 35 chars)

Returning End-to-End Transaction ID

bankTxnTime: string (up to 25 chars)

Returning HSBC Transaction time for the inward credit payment
