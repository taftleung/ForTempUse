Payment: object

id: string (up to 36 chars)

Unique Entity ID of a Payment

sqn: string (up to 36 chars)

A reference number for payment

merchant_id: string (up to 35 chars)

Merchant ID

master_erchant_id: string (up to 35 chars)

Master Merchant ID

amount: integer 1 â‰¤ x â‰¤ 999999999999

Payment Amount

  • Format: Eliminate punctuation and sign, support 2 decimal places according to ISO 4217, e.g. $10.50 = 1050
currency: string THB (up to 3 chars)

Order Currency

payer_bank_code: string (up to 3 chars)

Payer Bank Code

status: string SUCCESS, SETTLED (up to 20 chars)

Payment Status

txn_datetime: string (up to 25 chars)

Transaction time for the inward credit payment

  • Bank system local time. A GMT+7 timezone information is appended to the end of the timestamp to indicate this time is a Thailand local time. Format: yyyy-MM-dd'T'HH:mm:ssÂ±hh:mm
notify_datetime: string (up to 25 chars)

Notification time for the inward credit payment

  • Bank system local time. A GMT+7 timezone information is appended to the end of the timestamp to indicate this time is a Thailand local time. Format: yyyy-MM-dd'T'HH:mm:ssÂ±hh:mm
